    Enumerasi Sumber Daya di Mesin Lokal Menggunakan Hyena¶

    Hyena menggunakan tampilan seperti Windows Explorer untuk semua operasi, termasuk klik kanan menu konteks untuk semua obyek. Mengelola pengguna, grup (lokal dan global), berbagi, domain, komputer, services (layanan), perangkat, peristiwa, printer dan pekerjaan printer, sesi, membuka berkas, ruang diska, hak pengguna, perpesanan, pengeksporan, penjadwalan pekerjaan, proses dan percetakan semua didukung.

    Skenario Lab¶

    Hacker mengenumerasi aplikasi dan banner sebagai tambahan untuk mengidentifikasi akun pengguna dan sumber daya bersama. Di dalam lab ini, Hyena menggunakan tampilan seperti Windows Explorer untuk semua operasi, termasuk klik kanan menu konteks untuk semua obyek. Mengelola pengguna, grup (lokal dan global), berbagi, domain, komputer, services (layanan), perangkat, peristiwa, printer dan pekerjaan printer, sesi, membuka berkas, ruang diska, hak pengguna, perpesanan, pengeksporan, penjadwalan pekerjaan, proses dan percetakan semua didukung. Untuk menjadi Expert Ethical Hacker (Ahli Peretas Beretika) dan Penetration Tester (Penguji Penetrasi), Anda harus memiliki pengetahuan tentang enumerasi, dimana membutuhkan koneksi yang aktif ke mesin (komputer) yang sedang diserang.

    Tujuan Lab ¶

    Tujuan dari lab ini adalah untuk membantu peserta didik belajar dan melakukan enumerasi jaringan dari:

    • Informasi pengguna sistem
    • Menjalankan layanan sistem

    Lingkungan Lab ¶

    Untuk melakukan kegiatan lab ini, Anda membutuhkan:

    • Sebuah komputer yang menjalankan Windows Server 2012 atau Windows 7 dan lebih baru.
    • Hak administrator untuk memasang dan menjalankan alat.
    • Anda juga dapat mengunduh alat ini dari tautan: (https://www.systemtools.com/hyena/download.htm)
    • Jika Anda memilih untuk mengunduh versi terbaru dari alat ini, tangkapan gambar mungkin berbeda.

    Durasi Lab¶

    Waktu: 5 menit

    Gambaran dari Enumerasi ¶

    Enumerasi adalah proses mengektraksi nama pengguna, nama mesin, sumber daya jaringan, berbagi, dan layanan dari sebuah sistem. Teknik enumerasi dilakukan di dalam sebuah lingkungan intranet.

    Tugas Lab ¶

    1. Navigasi ke lokasi unduhan dari tool dan klik dua kali Hyena_English_x64.exe.
    2. Jika Open File - Security Warning tampil, klik Run.

    3. Wizard instalasi Hyena tampil, klik Next. Catatan: Jika Anda ditanya untuk memasang C++ Redirstributable, klik Install. Setelah pemasangan, jika membutuhkan memulai ulang sistem, klik Yes untuk memulai ulang mesin.

    4. Ikuti langkah-langkah untuk memasang Hyena.

    5. Pada saat penyelesaian pemasangan, bagian InstallShield Wizard complete tampil, klim Finish untuk menyelesaikan pemasangan.
    6. Setelah selesai memasang, jalankan aplikasi Hyena dari layar Apps (untuk Windows 8 dan Windows 8.1) atau Start Menu (untuk Windows 7 dan Windows 10).
    7. Jika SystemTools Update Notification Utility tampil, klik Close.
    8. Jika jendela Registration tampil, klik OK untuk melanjutkan.
    9. Jika dialog Hyena tampil, meminta Anda untuk mendaftarkan aplikasi, klik No.
    10. Jendela utama dari Hyena tampil, seperti yang ditampilkan di tangkapan gamber:
    11. Klik node + dari local workstation untuk meluaskan bagian, setelah itu buka Users node untuk melihat seluruh pengguna di mesin lokal.
    12. Untuk mengecek layanan yang berjalan di sistem, klik ganda Services .
    13. Klik ganda User Rights untuk menampilkan daftar hak pengguna.
    14. Klik ganda Scheduled Jobs untuk memeriksa perkejaan terjadwal.
    15. Dengan mengobservasi semua pilihan ini, Anda dapat mengecek untuk informasi wajar yang ditemukan oleh Hyena yang dapat meminta Anda untuk mengambil langkah keamanan yang benar untuk mengamankan sistem.
